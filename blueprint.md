# Lotto Number Generator

## Overview

A simple web application to generate random lottery numbers.

## Features

*   Generates 6 unique random numbers between 1 and 45.
*   Displays the numbers in a user-friendly interface.
*   A button to generate new numbers.

## Current Plan

*   **index.html**: Update the HTML structure for the lottery number generator.
*   **style.css**: Add styles for the lottery number generator.
*   **main.js**: Add JavaScript logic to generate and display the lottery numbers.
